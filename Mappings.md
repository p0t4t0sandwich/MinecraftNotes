# Mappings

Notes:
- "Intmdry" is short for "Intermediary"
- "SRG" is short for "Searge"
- "Hashed" is short for "Hashed Mojmap"
- "Official" mappings refer to the ProGuard'ed (obfuscated) code distributed in the official JAR
